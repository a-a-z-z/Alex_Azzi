# Netlify Deployment Guide for Alex Azzi Website

## Quick Deployment Steps

### Step 1: Open Netlify Drop

Go to: **https://app.netlify.com/drop**

### Step 2: Prepare Your Folder

Your website files are located at:

```
c:\Users\User\Desktop\New folder\alex-azzi-brand\
```

This folder contains:

- index.html
- css/style.css
- js/main.js
- README.md

### Step 3: Drag and Drop

1. Open File Explorer
2. Navigate to `c:\Users\User\Desktop\New folder\`
3. Find the `alex-azzi-brand` folder
4. **Drag the entire folder** onto the Netlify Drop page
5. Wait for upload to complete (usually 5-10 seconds)

### Step 4: Get Your URL

After upload completes, Netlify will show:

- A random URL like: `https://cheerful-unicorn-abc123.netlify.app`
- Copy this URL and share with your boss!

## Optional: Customize Your URL

If you want a better URL name:

1. Click "Site settings" after deployment
2. Click "Change site name"
3. Enter something like: `alex-azzi-brand`
4. Your new URL: `https://alex-azzi-brand.netlify.app`

## Troubleshooting

**If the site doesn't load:**

- Make sure you dragged the entire `alex-azzi-brand` folder
- Check that `index.html` is in the root of the folder
- Refresh the Netlify page and try again

**To update the site later:**

- Just drag the folder again to the same Netlify page
- It will automatically update
